
# 🚨Quero Denunciar

O Quero Denunciar foi criado para facilitar o registro de representações em órgãos municipais, estaduais, federais e particular. A Quero Denunciar não possui vinculo com órgãos ou empresas. É um sistema autonomo para registros de denúncias em âmbito nacional, sem nenhum patrocinador. O objetivo do sistema é facilitar as denúncias para todo o país, seja referente a denúncios públicas ou particulares, como abuso infantil, estupros, tráfico de drogas, abuso de autoridade, etc.
